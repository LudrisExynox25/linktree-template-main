📚 Master class Temas Oficiales por Niveles

🟢 Excel Básico (Los Cimientos)
  • Interfaz desde cero y navegación fluida por el programa.
  • Ingreso correcto de datos y operaciones matemáticas básicas.
  • Creación de gráficas visuales, uso de filtros y segmentación de datos.
🟡 Excel Intermedio (Herramientas de Productividad)
  • Control de datos con tablas dinámicas.
  • Dominio de fórmulas lógicas, de referencia y de búsqueda.
  • Introducción a las macros y la creación de facturas automatizadas.
🔴 Excel Avanzado (Automatización y Desarrollo)
  • Uso de funciones específicas para bases de datos.
  • Diseño y control de un sistema de inventario básico.
  • Programación con VBA: Creación de formularios de usuario (UserForms).
  • Construcción de Dashboards básicos y avanzados potenciados con código VBA.
